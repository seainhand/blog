AJAX那些事（一）基础

我们为什么要使用AJAX

想象这样一个注册用户的场景

当我们在注册用户的时候可能是这样一个表单

如果我们的注册用户名不符合规范，需要将全部input填写完并提交注册后才会提醒，但是此时我们已经浪费的大量的时间,这样的用户体验十分差。

我们需要的是在填写完用户名之后立即提醒用户名是否可以使用，如下图

AJAX正是完成此需求的最常用的手段,

什么是AJAX

AJAX 是一个简称（缩写），
分别由以下四个单词的首字母组成

Asynchronous         浏览器使用的异步通信，实现页面局部刷新(操作DOM树)

Javascript               使用的编程语言

And

Xml                          数据通信的承载方式，实际上很少使用XML

即使用javascript的编程语言，使用XML(JSON)等数据承载方式，实现异步通信，浏览器页面局部刷新。

我们如何使用AJAX

使用一个AJAX的流程为

  1. 创建承载访问和返回内容的对象，即AJAX的核心XMLHttpRequest对象
  • 告诉这个对象你要发请求到哪去，发的这个请求包含什么
  • 告诉这个对象你发成功了，返回了之后你要做些什么
  • 发送这个对象

分别用代码演示

创建XMLHttpRequest对象

//支持IE7+、以及其他主流浏览器
var xhr = XMLHttpRequest();
//支持全浏览器
var xhr = null;
if(window.XMLHttpRequest){
    xhr = new XMLHttpRequest();
} else if (window.ActiveXObject) {
    try{
        xhr = new ActiveXObject('Msxml2.XMLHTTP');
    } catch (e) {
        try{
            xhr = new ActiveXObject('Microsoft.XMLHTTP');
        }
    }
}

要注意的是XMLHttpRequest对象分level1和level2

老版本的XMLHttpRequest level有着一些缺点

  • 只支持文本数据的传送，无法用来读取和上传二进制文件。
  • 传送和接收数据时，没有进度信息，只能提示有没有完成。
  • 受到”同域限制”（Same Origin Policy），只能向同一域名的服务器请求数据。

新版本的XMLHttpRequest level2新增了很多功能

  • 可以设置HTTP请求的时限。
  • 可以使用FormData对象管理表单数据。
  • 可以上传文件。
  • 可以请求不同域名下的数据（跨域请求）。
  • 可以获取服务器端的二进制数据。
  • 可以获得数据传输的进度信息。
    　　
    当使用level2时请注意浏览器的兼容问题

设定请求包含内容

open方法进行初始化操作，初始请求的一些信息，使用open方法后，XMLHttpRequest对象变为open状态，才可以进行发送。

open方法包含五个参数

xhr.open(
    //请求提交的方式，一般用GET方法,或POST方法
    'GET', 

    //请求的目标，即URL，使用GET方法是将传递的参数写入URL中
    'http://seainhand.github.io/test?user=lixiang', 

    //确定请求的为同步或者异步，异步为true,同步为false，正常情况下为true
    true, 

    //用户名，当服务器需要用户名密码验证的时候使用
    'username', 

    //密码，当服务器需要用户名密码验证的时候使用
    'password', 
);

setRequestHeader方法给请求加入头部信息

//给请求头加入Content-Type信息
xhr.setRequestHeader('Content-Type','application/json');

//给请求头加入自定义的user信息
xhr.setRequestHeader('user','lixiang');

设定请求完成后进行的动作

XMLHttpRequest有一个名为readystate的属性，它表示XMLHttpRequest对象的五种状态，即

  • 0:未初始化
  • 1 : 连接建立，请求发出
  • 2 : 服务器返回相应
  • 3 : 交互（处理相应数据）
  • 4 : 完成，数据可交付客户端使用

只有当readystate等于4,并且需要http的状态码为200时，我们才有必要处理我们的逻辑代码，即

if (xhr.readyState === 4 && xhr.status == 200) {

    //获取返回的文本
    console.log(xhr.responseText);

    //将字符串转化为JSON对象
    JSON.parse(xhr.responseText);
    eval(xhr.responseText);

    //获取返回的XML        
    console.log(xhr.responseXML);

    //获取响应的头部信息
    console.log(xhr.getAllResponseHeaders());

    //获取响应的某些头部信息
    console.log(xhr.getResponseHeader("Content-Type"));

    //获取响应的状态码
    console.log(xhr.status);

    //获取响应的状态文本
    console.log(xhr.statusText);
}

发送请求

当XMLHttpRequest对象为open状态时可以使用send方法发送请求

//当提交方式为POST方法时，将参数写入send方法中，使用urlencoded模式，即用&链接的key=value值
xhr.send('user=lixiang&age=20&sex=male'); 

//JSON模式,HTTP请求只能发送文本，所以使用JSON.stringfy方法进行序列化
xhr.send(JSON.stringfy({
    user : lixiang,
    age : 20,
    sex : male
}));

这样我们就完成了一个AJAX过程
